# Prague.Api

> Part of **[Prague](https://github.com/nanov/prague.net)** — a high-performance, compile-time-safe
> in-memory event cache for .NET, fed by event-sourced Kafka streams.

An HTTP inspection surface for live Prague caches — browse registered caches, inspect indexes, and
run queries against the in-memory state for debugging and operations. Pair with `Prague.Api.UI` for a
browser front end.

## Install
`dotnet add package Prague.Api`

See the [project README](https://github.com/nanov/prague.net) for the full guide.
